# Images Marketplace - A fournir par l'infographiste

Toutes les images doivent etre placees dans ce dossier : `public/images/marketplace/`

## Hero Banner
| Fichier | Description | Dimensions | Format |
|---------|------------|------------|--------|
| `cat_hero_banner.png` | Image hero desktop : sacs shopping colores sur fond sombre (vert/teal), style premium marketplace | 1200x500 | PNG transparent ou JPEG |
| `cat_hero_mobile.png` | Version mobile du hero (optionnel, sinon le desktop sera crop) | 600x400 | PNG |

## Icones Categories (style flat/outline, fond transparent)
Chaque icone represente une categorie de produits. Style : minimaliste, monochrome blanc ou vert emeraude.

| Fichier | Description | Dimensions |
|---------|------------|------------|
| `cat_alimentation.png` | Panier de courses / fruits | 64x64 |
| `cat_cosmetique.png` | Flacon de parfum / pinceau | 64x64 |
| `cat_electronique.png` | Smartphone / ecouteurs | 64x64 |
| `cat_vetements.png` | T-shirt / cintre | 64x64 |
| `cat_informatique.png` | Laptop / clavier | 64x64 |
| `cat_papeterie.png` | Cahier / stylo | 64x64 |
| `cat_maison.png` | Maison / meuble | 64x64 |
| `cat_autre.png` | Etoile / boite cadeau | 64x64 |
| `cat_default.png` | Icone generique (sac shopping) | 64x64 |
