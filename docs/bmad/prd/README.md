# PRD (Product Requirements Documents)

Ce dossier contient les documents de spécifications produit BMAD.

## Structure des fichiers

```
prd/
├── PRD_MAIN.md              # PRD principal du projet (existant)
└── PRD_{FEATURE}.md         # PRD par fonctionnalité (évolutions)
```

## Convention de nommage

- `PRD_MAIN.md` : Document principal décrivant le projet global
- `PRD_{NOM_FEATURE}.md` : Document pour chaque nouvelle fonctionnalité
  - Exemple : `PRD_SOCIAL_COMMERCE.md`, `PRD_AI_RECOGNITION.md`

## Template

Utiliser le template : `.claude/bmad/templates/PRD.tpl.md`

## Workflow

1. Copier le template PRD.tpl.md
2. Renommer en `PRD_{FEATURE}.md`
3. Remplir les sections avec les spécifications
4. Faire valider par le Product Manager Agent
5. Passer à la phase Architecture
